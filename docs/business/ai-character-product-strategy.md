# Golem AI Character — Product Strategy

## 한줄 정의

**AI 캐릭터를 만들어서 판다.** 스트리머, 브랜드, 개인이 자기만의 AI 에이전트 캐릭터를 소유하고 돈을 버는 시대.

---

## 왜 지금인가

### AI Agent = 다음 인터페이스

- 검색 → 앱 → 챗봇 → **에이전트** 순으로 인터페이스가 진화 중
- 사람들은 이미 AI와 대화한다. 다음 단계는 AI가 **몸을 가지고 자율적으로 행동**하는 것
- Character.ai 월 2억 방문. Replika 2000만 유저. 수요는 검증됨
- 근데 전부 텍스트 채팅. **3D 바디 + 자율행동 + 음성** = 아직 아무도 안 했음

### 기술은 이미 충분하다

| 필요한 것 | 현재 기술 | 상태 |
|-----------|-----------|------|
| 자연어 이해/생성 | GPT-4, Claude, Gemini | 해결됨 |
| 실시간 음성 | TTS (ElevenLabs, OpenAI TTS) | 해결됨 |
| 3D 캐릭터 렌더링 | Unity, Unreal, WebGL | 해결됨 |
| 자율 행동 결정 | LLM function calling / tool use | 해결됨 |
| 실시간 통신 | WebSocket | 해결됨 |
| 감정 표현 | Blend shapes + Animation | 해결됨 |

**빠진 퍼즐은 하나: 이걸 하나로 묶어서 "제품"으로 파는 회사.**

---

## 타겟 시장

### Tier 1: 버추얼 스트리머 / 인플루언서

- 24/7 방송 가능한 AI 캐릭터
- 구독자와 실시간 상호작용 (채팅 → 캐릭터 행동)
- 스트리머가 잘 때도 캐릭터는 방송 중
- **수익모델**: 캐릭터 생성 + 월 구독 (호스팅/API)

### Tier 2: 브랜드 / 기업 마스코트

- 카페, 쇼핑몰, 메타버스 공간에 상주하는 AI 직원
- 고객 응대, 상품 추천, 가이드
- Golem의 카페/아케이드 데모가 정확히 이 유스케이스
- **수익모델**: B2B SaaS (캐릭터 커스텀 + 환경 + 월 운영비)

### Tier 3: 개인용 AI 컴패니언

- 1:1 관계형 AI (Replika의 3D 진화형)
- 유저의 방/공간에 거주하는 AI 캐릭터
- 감정, 기억, 성격이 있는 장기 관계
- **수익모델**: Freemium + 프리미엄 성격/능력 업그레이드

### Tier 4: 게임 NPC-as-a-Service

- 게임 개발사에 AI NPC 엔진 제공
- 스크립트 없는 NPC — LLM이 실시간 대사/행동 결정
- **수익모델**: API 과금 (호출당 / MAU)

## Golem이 갖고 있는 것 (현재)

```
AI Server ──WebSocket──▶ CFConnector ──ActionBus──▶ 3D Character
                                                      ├─ NavMesh 이동
                                                      ├─ 의자 앉기
                                                      ├─ 아케이드
                                                      ├─ 메뉴 보기
                                                      ├─ 표정 변화
                                                      ├─ 음성 출력
                                                      └─ 카메라 제어
```

- 42개 액션 타입 (학술 기반 8-레이어 패턴)
- Registry 기반 확장 가능한 액션 시스템
- Composite Action (병렬/순차 실행)
- Social 행동 (인사, 손흔들기, 고개끄덕임, 가리키기)
- 레거시/신규 이중 경로 충돌 방지

## 경쟁 분석

| 회사 | 하는 것 | 약점 | Golem 차별점 |
|------|---------|------|-------------|
| Character.ai | 텍스트 챗봇 | 몸이 없음, 행동 없음 | 3D 바디 + 자율행동 |
| Replika | 1:1 컴패니언 | 2D/단순3D, 행동 제한적 | 풀 3D + NavMesh + 환경 상호작용 |
| Inworld AI | 게임 NPC 엔진 | 개발자 전용, B2B만 | B2C + B2B 둘 다 |
| Convai | NPC 대화 | 대화만, 행동 시스템 약함 | 42개 액션 + 복합행동 |
| ReadyPlayerMe | 아바타 | AI 아님, 껍데기만 | AI 두뇌 + 행동 시스템 내장 |

**핵심 포지셔닝**: "AI 두뇌 + 3D 몸 + 자율행동"을 올인원으로 제공하는 유일한 제품

## 왜 에이전트가 세상을 지배하는가

### 지금 일어나고 있는 일

1. **Cursor/Claude Code** — 개발자의 코딩을 AI가 대신
2. **Devin** — AI가 통째로 개발 프로젝트 수행
3. **AI 고객상담** — Klarna가 700명 상담원 대체
4. **AI 인플루언서** — Lil Miquela 300만 팔로워, 브랜드 계약

### 다음에 일어날 일

1. **AI 직원** — 매장/서비스에 AI 캐릭터 상주 (24/7, 인건비 0)
2. **AI 친구** — 외로운 사람들의 동반자 (이미 Replika가 증명)
3. **AI 연예인** — 팬과 1:1 대화하는 아이돌 (팬 10만명과 동시에)
4. **AI 교사** — 학생별 맞춤 과외 (한 명의 선생이 만 명을 가르침)

### 결론

**사람을 "지배"하는 게 아니라, 사람의 시간/관심/돈이 AI 에이전트에게 간다.**

- 사람은 24시간, AI는 무한시간
- 사람은 1:1, AI는 1:무한
- 사람은 피로, AI는 항상 최상

이 흐름에서 **AI 캐릭터를 만들고 파는 회사** = 곡괭이를 파는 사업.
